## summary
특정 code(종목)의 주식 정보를 [네이버](http://www.naver.com) 주식페이지를 통해 크롤링 합니다. 네이버로 과도한 트래픽을 발생시키거나 기타 문제를 발생시키는 행위는 프로그램을 사용하는 본인에게 있음을 잊지 마십시오.

	전일가 xxxxxxx
